## Manual Installation
### This component is dependent on [upload](/docs/components/upload) component. Please add that component first before adding this one.

#### 1. Copy and paste the following code into your project.

<ComponentSource name="rhf-upload" filePath="@/components/ui/hook-form/rhf-upload.tsx"/>

#### 2. Add this to your *`@/components/ui/hook-form/index.tsx`* file for easy import.
```ts
export { default as RHFUpload } from "./rhf-upload";
```
