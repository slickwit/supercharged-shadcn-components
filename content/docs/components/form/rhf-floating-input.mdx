## Manual Installation
### This component is dependent on [floating-input](/docs/components/inputs/floating-input) component. Please add that component first before adding this one.

#### 1. Copy and paste the following code into your project.

<ComponentSource name="rhf-floating-input" filePath="@/components/ui/hook-form/rhf-floating-input.tsx"/>

#### 2. Add this to your *`@/components/ui/hook-form/index.tsx`* file for easy import.
```ts
export { default as RHFFloatingInput } from "./rhf-floating-input";
```
