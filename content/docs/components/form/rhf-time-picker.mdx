## Manual Installation
### This component is dependent on [time-picker](/docs/components/inputs/time-picker) and [floating-button](/docs/components/buttons/floating-button) components. Please add that component first before adding this one.

#### 1. Copy and paste the following code into your project.

<ComponentSource name="rhf-time-picker" filePath="@/components/ui/hook-form/rhf-time-picker.tsx"/>

#### 2. Add this to your *`@/components/ui/hook-form/index.tsx`* file for easy import.
```ts
export { default as RHFTimePicker } from "./rhf-time-picker";
```
