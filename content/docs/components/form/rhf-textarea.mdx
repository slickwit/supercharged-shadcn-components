## Manual Installation
### This component is dependent on [floating-textarea](/docs/components/inputs/floating-textarea) component. Please add that component first before adding this one.

#### 1. Copy and paste the following code into your project.

<ComponentSource name="rhf-textarea" filePath="@/components/ui/hook-form/rhf-textarea.tsx"/>

#### 2. Add this to your *`@/components/ui/hook-form/index.tsx`* file for easy import.
```ts
export { default as RHFTextarea } from "./rhf-textarea";
```
